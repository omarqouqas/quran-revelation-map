## Style Guidelines (Use in ALL prompts)

Add these modifiers to maintain consistent visual language:

```
Style: cinematic, documentary film, reverent atmosphere, 4K
Color palette: warm golds, deep blues, desert earth tones
Mood: awe-inspiring, peaceful, spiritually evocative
Camera: slow pan, gentle movement, steady
Avoid: modern elements, faces, cartoonish style, text overlays
```

## 4. GENERATION WORKFLOW

1. Create account on Runway/Pika
2. Start with ambient backgrounds (5 videos)
3. Generate at 16:9 aspect ratio, 5-10 seconds

### Step 2: Generate Ambient Backgrounds
```bash
# File naming convention
dawn.mp4      # 610 CE
morning.mp4   # 614 CE
midday.mp4    # 621 CE
afternoon.mp4 # 628 CE
```

## 5. QUALITY TIPS

1. **Generate multiple versions** - AI is inconsistent, generate 3-5 and pick best
2. **Use seed locking** - If you find a good frame, lock the seed for variations
3. **Upscale after** - Generate at base quality, then upscale with Topaz or similar
4. **Loop seamlessly** - Use tools like Runway's extend feature to create seamless loops
5. **Color grade** - Apply consistent color grading in post for cohesive look
